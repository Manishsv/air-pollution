These documents are retained for historical reference but are **not current guidance**. They fall into one of three categories:

## Historical / superseded

- `AIR_OS_ARCHITECTURE_REVIEW_2026_05_02.md` — architecture review from 2026-05-02, references removed `src/` package
- `AIR_OS_ARCHITECTURE_CHECKPOINT_2026_05_02.md` — same session, same caveats
- `AIR_OS_CODE_REVIEW_TRIAGE_2026_05_06.md` — triage classification, useful for historical tracking only
- `AIR_PLATFORM_ARCHITECTURE.md` — earlier production vision (Keycloak/Redis/Kong) that diverged from current pilot runtime
- `REPO_RESTRUCTURING_PLAN.md` — restructuring plan; Phase 3 (SDK) is done, rest deferred
